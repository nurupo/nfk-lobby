# NFK Lobby

NFK Lobby is a lobby application for a video game called [Need For Kill](http://info.needforkill.ru/).

It mainly consists of three major parts:

* Planet Scanner — lists current network games.
     
* Chat — a simple single-server implementation of IRC client.
     
* Game Preferences — allows players to share what gametypes they currently want to play.
